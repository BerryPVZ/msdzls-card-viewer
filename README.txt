MeiShi Card Viewer
===================

Open index.html in Chrome, Edge, Firefox, or Opera. No web server is required.

Catalog rows: 14642
Unique IDs: 14629
Included card PNGs: 969

The site defaults to entries that have PNG artwork. Turn off “Only show entries with card PNGs” to browse the complete card_desc catalog.
Keyboard: / focuses search, Escape closes details, Left/Right moves through open results.
